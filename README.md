# Flutter Custom Chat UI

Hi! The goal of project is to provide the custom flutter chat ui with beautiful design. This is pure front-end based coding with flutter. 


# UI Screens

There are three simple beautiful screens.
- **Splash Screen**
-  **Recent Chat Screen**
-  **Chat Screen**
